Résumé

Ce projet vise à développer un système complet de gestion des stages pour les étudiants, en utilisant une approche de conception basée sur la méthode Merise et une implémentation technique avec les frameworks Spring Boot (backend) et Angular (frontend).

Conception avec Merise : La conception a été réalisée en utilisant la méthode Merise, une méthodologie de modélisation de systèmes d’information. Les entités clés identifiées incluent Stage, Etudiant, Professeur, Entreprise, et Promotion. Les relations entre ces entités ont été spécifiées avec des cardinalités et des contraintes, permettant une représentation claire des interactions dans le système.

Implémentation Technique : L’implémentation technique s’appuie sur Spring Boot pour la partie backend, fournissant une architecture robuste basée sur Java. Les entités définies dans la conception Merise sont traduites en classes Java annotées avec JPA pour la persistance des données dans une base PostgreSQL.

La partie frontend est développée avec Angular, un framework JavaScript populaire. L’interface utilisateur offre une expérience conviviale pour les utilisateurs, permettant la gestion des stages, des étudiants, des professeurs, des entreprises, et des promotions.
